# Sellable Safe Edition

Deze map is bedoeld als verkoopklare, veilige editie van de webshop.

Doel:

- geschikt om als product/template aan andere server-eigenaren te verkopen;
- geen experimentele RP-bundels;
- geen geld, zwart geld of wapens als standaardproducten;
- focus op supporter ranks, cosmetics, priority queue en veilige donateurvoordelen.

## Wat zit erin?

- `PRODUCT_POLICY.md` - uitleg welke producten veilig zijn en welke niet worden meegeleverd.
- `SAFE_PRODUCTS.md` - voorbeeld productcatalogus voor verkoopklare versie.
- `CUSTOMER_INSTALL_GUIDE.md` - klantvriendelijke installatiehandleiding.
- `DELIVERY_COMMANDS.md` - veilige delivery command voorbeelden.
- `LICENSE_TEMPLATE.md` - eenvoudige licentie/gebruikstekst die je kunt aanpassen.

## Aanbevolen verkoopnaam

```txt
FiveM Safe Stripe Webshop Bridge
```

of:

```txt
RP Server Stripe Store Bridge - Safe Edition
```

## Aanbevolen verkoopomschrijving

Een veilige Next.js/Vercel webshop met Stripe iDEAL/card betalingen, MySQL order queue en FiveM bridge zonder extra serverpoorten. Ontworpen voor ranks, cosmetics, priority queue en supporter perks.

## Niet meegeleverd in Safe Edition

- pay-to-win geldpakketten;
- zwart geld delivery;
- wapenpakketten;
- random lootboxes/crates;
- experimentele producten.

## Aanbevolen klantbelofte

Deze template helpt server-eigenaren een professionele supporter webshop op te zetten. De klant blijft zelf verantwoordelijk voor naleving van de actuele FiveM/Cfx regels, betaalproviderregels en lokale wetgeving.
