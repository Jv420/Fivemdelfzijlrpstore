# HexTactics Store Bridge Pro

Deze map is de verkoopklare HexTactics-basis voor de webshop/script combinatie.

Doel:

- van het project een verkoopbaar product maken;
- HexTactics als hoofdmerk gebruiken;
- Delfzijl RP/DelfzijlCity/Minecraft gescheiden houden als eigen projecten;
- een veilige, professionele editie aanbieden aan klanten.

## Productnaam

```txt
HexTactics Store Bridge Pro
```

## Korte pitch

Een professionele FiveM webshop bridge met Next.js, Vercel, Stripe iDEAL/card, MySQL/MariaDB en automatische FiveM delivery zonder extra inkomende poorten.

## Verkoopfocus

Deze editie richt zich op:

- supporter ranks;
- priority queue;
- cosmetic perks;
- custom kenteken;
- vehicle cosmetics;
- donateur garage toegang;
- veilige order delivery;
- duidelijke documentatie.

## Niet in de verkoopeditie

- geldpakketten;
- zwart geld;
- wapens;
- pay-to-win bundels;
- experimentele producten.

## Structuur

- `BRANDING.md` - merknaam, positionering en footer teksten.
- `SALES_TEXT.md` - verkooptekst voor webshop/Tebex/CFX/GitHub.
- `PRICING.md` - prijsadvies en pakketten.
- `ROADMAP.md` - roadmap naar v1.0 en v2.0.
- `RELEASE_CHECKLIST.md` - checklist voordat je gaat verkopen.
- `SUPPORT_POLICY.md` - supportregels voor klanten.
- `CHANGELOG.md` - changelog template.
