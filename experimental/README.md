# Experimental producten

Deze map is bewust los gehouden van de normale webshop.

Doel:

- ideeën voor starterbundels bewaren;
- producten voorbereiden zonder ze automatisch actief te maken;
- duidelijk scheiden tussen veilige/compliant shopproducten en experimentele RP-producten.

## Belangrijk

De bestanden in deze map worden standaard niet gebruikt door `app/page.tsx`, `lib/products.ts` of de Stripe checkout.

Activeer dit alleen als je zelf hebt gecontroleerd dat het past binnen de actuele FiveM/Cfx regels en jouw serverbeleid.

## Inhoud

- `products.experimental.ts` - voorbeeldproducten voor starterbundels en RP-bundels, standaard uitgeschakeld.
- `delivery-commands.example.lua` - voorbeeld van staff/admin server commands die je handmatig kunt gebruiken of later gecontroleerd kunt koppelen.

## Veilig advies

Gebruik de hoofdshop voor:

- ranks;
- priority queue;
- cosmetics;
- garage toegang;
- custom kenteken;
- voertuigskins.

Gebruik deze experimental map alleen als klad/voorbereiding.
